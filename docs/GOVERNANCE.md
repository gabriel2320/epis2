# Governance

EPIS2 was reset to avoid returning to an inflated project.

## Complexity Budget

- One web app.
- One API.
- One OpenAPI contract.
- One local infrastructure stack.
- Five official gates: API, web, contract, e2e, full.

## Forbidden Growth

- No CICA revival.
- No `/espacio` compatibility layer.
- No dashboard as home.
- No labs attached to core.
- No historical gates catalog.
- No second command or form registry.
- No broad helper folders without domain ownership.

## Feature Ladder

Before adding anything, answer:

1. Does it belong to patient, ficha, SOAP, audit, print, permissions, or optional AI?
2. Does the backend own the clinical truth?
3. Does it have a minimal human flow?
4. Does it have a test?
5. Can an existing module absorb it without abstraction?

If any answer is no, keep it out of core.
